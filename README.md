# 🤖 Advanced Rule-Based Chatbot (Python)

A rule-based chatbot built using Python that simulates intelligent conversation using `if-elif-else` statements.  
This project demonstrates basic Natural Language Processing structure without using Machine Learning.

---

## 🚀 Features

- Interactive console chatbot
- Remembers user's name
- Tells random programming jokes
- Shows current date & time
- Performs basic math calculations
- Provides random smart responses
- Exit command support
- Uses simple NLP techniques

---

## 🛠️ Technologies Used

- Python
- datetime module
- random module
- Basic NLP (string matching)



## 🧠 How It Works

The chatbot:

1. Uses an infinite `while` loop
2. Takes user input
3. Converts input to lowercase
4. Matches keywords using `if-elif-else`
5. Responds accordingly
6. Stops when user types `bye`

It also:
- Stores user name in a variable
- Uses `random.choice()` for varied replies
- Uses `datetime` for date & time
- Uses `eval()` for simple calculator functionality

---

## 💬 Example Conversation

```
You: hello
Bot: Hello! Nice to meet you.

You: my name is abhishek
Bot: Nice to meet you, Abhishek!

You: who am i
Bot: You are Abhishek!

You: 5 + 7
Bot: The answer is 12

You: tell me a joke
Bot: Why do programmers prefer dark mode? Because light attracts bugs!
```

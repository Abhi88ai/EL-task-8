# 🤖 Advanced AI Chatbot (Python)

An AI-powered conversational chatbot built using Python and HuggingFace Transformers.  
This chatbot uses a pre-trained language model to generate intelligent, human-like responses.

---

## 🚀 Features

- AI-based conversation (not rule-based)
- Uses Microsoft DialoGPT model
- Maintains conversation context
- Natural language understanding
- Exit command support
- Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python
- Transformers (HuggingFace)
- PyTorch
- Pre-trained NLP Model (DialoGpt)

## 🧠 How It Works

1. Loads pre-trained DialoGPT model
2. Tokenizes user input
3. Generates response using deep learning
4. Maintains conversation history
5. Prints AI-generated reply

---

## 💬 Example Conversation

```
You: Hello
Bot: Hi there! How are you today?

You: I am feeling stressed
Bot: I'm sorry to hear that. Want to talk about it?
```
